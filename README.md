Course Project

ОПИС ПРОЄКТУ
------------
Course Project — це настільний застосунок для автоматизації основних процесів
роботи книжкового магазину. Застосунок розроблений для ведення обліку товарів,
категорій, клієнтів і працівників, оформлення продажів та закупівель.

Проєкт розроблений у межах курсової роботи з метою закріплення практичних навичок
роботи з платформою .NET, базами даних та побудови графічного інтерфейсу
користувача.


ВИКОРИСТАНІ ТЕХНОЛОГІЇ
-------------------------------------------------------------
- Мова програмування: C#
- Платформа: .NET (Windows Forms)
- База даних: MySQL
- IDE: Microsoft Visual Studio 2022
- Система контролю версій: Git, GitHub
- Зовнішні бібліотеки: MySql.Data (через NuGet)

АРХІТЕКТУРА
-------------------------------------------------------------
Проєкт реалізований з використанням принципів розділення відповідальностей.
Моделі використовуються для опису сутностей предметної області, модулі Data та
Database відповідають за взаємодію з базою даних, а Forms і Pages реалізують
користувацький інтерфейс. Допоміжна логіка винесена в окремі утилітні класи,
що спрощує підтримку та розширення проєкту.


ЗАПУСК ПРОЄКТУ
-------------------------------------------------------------
1. Встановити MySQL Server (рекомендовано використовувати MySQL Workbench).
2. Створити базу даних та виконати SQL-скрипт зі структурою БД.
3. Відкрити файл Course-Project.sln у середовищі Visual Studio.
4. Перевірити та за потреби змінити рядок підключення до бази даних.
5. Зібрати та запустити застосунок.
