Course Project

ОПИС ПРОЄКТУ
------------
Course Project — це desktop застосунок для автоматизації роботи книжкового магазину.
Застосунок дозволяє керувати товарами, категоріями, клієнтами, працівниками,
замовленнями, закупівлями та формувати аналітичні звіти. 
Проєкт розроблений у межах курсової роботи та реалізований на платформі .NET
з використанням Windows Forms і бази даних MySQL.


ВИКОРИСТАНІ ТЕХНОЛОГІЇ
-------------------------------------------------------------
- Мова програмування: C#
- Платформа: .NET (Windows Forms)
- База даних: MySQL
- IDE: Microsoft Visual Studio 2022
- Контроль версій: Git, GitHub
- Пакети: MySql.Data (через NuGet)

АРХІТЕКТУРА
-------------------------------------------------------------
Проєкт побудований за принципами розділення відповідальностей:
- Models відповідають за опис сутностей;
- Data та Database — за роботу з базою даних;
- Forms та Pages — за інтерфейс користувача;
- Utils — за спільну допоміжну логіку.

ЗАПУСК ПРОЄКТУ
-------------------------------------------------------------
1. Встановити MySQL Server.
2. Створити базу даних та виконати SQL-скрипт зі структури БД.
3. Відкрити файл Course-Project.sln у Visual Studio.
4. Перевірити рядок підключення до БД.
5. Зібрати та запустити проєкт.
